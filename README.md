  z-index: 1;

  z-index: 10;
}

      viewport={{ once: true }}

## 🐍 Contribution Snake Animation

Below is a live visualization of my contribution graph animated as a retro snake arcade game:

<p align="center">
  <img src="https://github.com/Dhanush-kumar-m/Dhanush-kumar-m/raw/output/github-contribution-grid-snake.svg" alt="GitHub Contribution Grid Snake" width="100%" />
</p>

### 🛠️ GitHub Actions workflow Setup
To generate this snake animation daily, you can add this workflow file to your profile repo:

Create a file named `.github/workflows/snake.yml` with the following configuration:
```yaml
name: Generate Snake Animation

on:
  schedule: # Runs every 12 hours
    - cron: "0 */12 * * *"
  workflow_dispatch: # Allows manual trigger
  push:
    branches:
      - main

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    
    steps:
      # Generates a snake game from a github user contributions graph
      - name: Generate github-contribution-grid-snake.svg
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
            
      # Push the content of <build_dir> to a branch
      - name: Push github-contribution-grid-snake.svg to Output Branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
